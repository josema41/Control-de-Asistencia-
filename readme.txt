# Sistema de Asistencia con Código QR 🚀

Este proyecto es una aplicación web desarrollada con **Streamlit** para gestionar la asistencia mediante el escaneo de códigos QR utilizando **Python** y **OpenCV**.

## 🛠️ Tecnologías utilizadas
* [Python 3.x](https://www.python.org/)
* [Streamlit](https://streamlit.io/) - Para la interfaz web.
* [OpenCV](https://opencv.org/) - Para el procesamiento de imágenes y lectura de QR.
* [SQLite](https://www.sqlite.org/) - Base de datos local.

## 📁 Estructura del Proyecto
* `asistencia.py`: El tablero (dashboard) principal donde se visualizan los registros.
* `escaner.py`: El script encargado de abrir la cámara y decodificar los QR.
* `database.db`: Base de datos donde se almacenan las asistencias.
